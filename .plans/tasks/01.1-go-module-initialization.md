## Scope (this sub-task only)
Initialize the Go module under the correct module path. This sub-task covers only the deliverables listed below; other sub-tasks of plan 01 cover the remaining scope of the parent plan.

## Deliverables
- Create directory `gobra-go/`.
- Run `go mod init github.com/viperproject/gobra`.
- Define standard Go version in `go.mod` (minimum 1.21).

For full context see parent plan [01-project-setup.md](../01-project-setup.md) §Deliverables.

## C9
Implement all Gobra annotations from parent plan §Verification Specifications (C9) in [01-project-setup.md](../01-project-setup.md) whose functions/types are defined in this sub-task's deliverables.

## Done when
- Verify `go.mod` exists with the correct package path and compiles cleanly.
