# Sub-task 15b.2 — Worker Process Lifespans
**Parent plan:** [15b-worker-pool-expansion.md](../15b-worker-pool-expansion.md)
**Read first:** §Scope, §Deliverables, §Verification Specifications (C9) in [15b-worker-pool-expansion.md](../15b-worker-pool-expansion.md)

## Scope (this sub-task only)
Ensure worker goroutines manage dedicated JVM subprocess processes. This sub-task covers only the deliverables listed below; other sub-tasks of plan 15b cover the remaining scope of the parent plan.

## Deliverables
- Assign a separate subprocess process to each worker and guarantee cleanups on pool stop.

For full context see parent plan [15b-worker-pool-expansion.md](../15b-worker-pool-expansion.md) §Deliverables.

## C9
Implement all Gobra annotations from parent plan §Verification Specifications (C9) in [15b-worker-pool-expansion.md](../15b-worker-pool-expansion.md) whose functions/types are defined in this sub-task's deliverables.

## Done when
- Assert that stopping the pool kills all associated JVM subprocesses.
